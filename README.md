  #### 前言

本项目由 Vite + Vue3 + TypeScript  + OpenIMServer 搭建

### 目标功能

- [x] 登录
- [x] 注册
- [x] 国际化
- [x] 单聊功能
- [x] 群聊功能
- [x] 发送表情、文件、图片、视频、好友名片、聊天记录、消息引用
- [x] 好友关系管理、黑名单
- [x] 个人信息管理以及头像上传
- [x] 置顶聊天、免打扰、已读消息
- [x] 添加好友、添加群聊以及创建群聊
- [x] 掉线重连

### 项目构建


```
# install dependencies
npm install 

# serve with hot reload at localhost:3000 
npm run dev 

# build for production with minification 
npm run build 

# Lint with EsLint

npm run lint

```
